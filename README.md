# SkillsJava - Learning Path

This repository is dedicated to practical learning of Java and Spring Boot, focusing on specific "Skills" through mini-projects.

## Skills Roadmap

1.  **Java Fundamentals (Java-Fundamentals)**
    *   Basic Syntax
    *   Object-Oriented Programming (OOP)
    *   Date and Time API (JSR-310)
    *   Collections API
    *   Stream API
    *   Records (Java 16+)
    *   Sealed Classes (Java 17+)
2.  **Spring Boot** (Next Step)

## Environment
- **Java Version:** 21
- **Dependency Manager:** Maven (recommended)
